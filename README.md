# DCsimpler_mv3 
AI를 사용하여, dcsimpler 확장 프로그램을 mv3를 지원하도록 업데이트했습니다.  
하는 김에 기존에 쓰이던 jQuery와 자바스크립트를 타입스크립트와 WXT로 대체했습니다.  
레이아웃 기능만 사용하기 때문에, 레이아웃이 제대로 나오는지만 테스트했고,  
나머지 기능이 제대로 작동하는지는 테스트 안 했습니다.  

## 설치 방법
릴리즈된 압축파일을 압축 해제한 다음,  
설치 방법은 브라우저 확장프로그램의 개발자 모드를 키시고,  
"압축 해제된 확장 프로그램 로드"를 눌러 압축 해제한 폴더를 선택하시면 됩니다.  

## 개선 사항
미니 갤러리도 지원.  
설정 버튼 삭제.  
"세부 설정" 사이드 메뉴에 로그인 버튼 삭제.  
자동 새로고침을 설정하면 모든 갤러리에서 유지되도록 변경.  
로딩 현황 등등의 아이콘 간격 조정.  
갤러리 별 차단 기능 구현.  
제목 키워드로 차단된 게시물에서, 걸린 키워드를 빨간색으로 강조.  
화면 우측 하단의 TOP · 차단 글 보기 버튼 비활성화 기능 추가.  
DC 웹페이지 로드 속도가 제 체감상 빨라진 것 같습니다.
